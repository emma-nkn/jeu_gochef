# GoChef

GoChef est un jeu incrémental où vous passez de commis de cuisine à propriétaire d'un empire gastronomique, directement depuis votre navigateur.

## Concept du Jeu

Le but est de préparer un maximum de plats pour accumuler des Pièces d'Or. Ces pièces permettent d'acheter du matériel de cuisine plus performant et d'embaucher du personnel pour aller vite en production.

### Logique de Progression

Le jeu repose sur deux piliers mathématiques :

Production Manuelle (Le Clic) : Chaque clic sur l'icône du plat principal génère une valeur de base de 1 pièce. Cette valeur peut être augmentée via des ustensiles.

Production Automatique (Le Tick) : Le jeu calcule chaque seconde la somme de la production de tous vos actifs (cuisiniers, robots, fours).

Formule du prix : Le coût de chaque amélioration augmente de 15% à chaque achat.
